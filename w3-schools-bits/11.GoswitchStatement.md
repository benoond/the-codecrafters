# Go switch Statement
Switch statement in Go is a cleaner, more readable version of multiple if-else statements.
Switch is either Single-Case or Multi-Cases

## Single-Case switch
Syntax:
```go
switch expression {
case value1:
    // code if expression == value1
case value2:
    // code if expression == value2
default:
    // code if no case matches
}
```

## Multi-Cases switch
Syntax for Multi-Cases
```go
switch expression {
case value1, value2, value3:
    // code if expression matches any of these values
case value4, value5:
    // code if expression matches any of these values
default:
    // code if no case matches
}
```
