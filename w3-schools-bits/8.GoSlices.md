# Go Slices
A slice is a more flexible and powerful abstraction built on top of arrays. Unlike arrays, slices can grow or shrink dynamically, and you don’t need to know the size upfront.

## Slice Declaration
This example declares two slice (slice1 and slice2):
```go
var slice1 = []int{1,2,3}
slice2 := []int{4,5,6,7,8}
```

## Modify Slice
Slice is modify by Append and Copy.

### Append Elements To a Slice
Example

```go
myslice1 := []int{1, 2, 3, 4, 5, 6}
myslice1 = append(myslice1, 20, 21)
```
### Copy Elements To a Slice
```go
src := []int{1, 2, 3, 4, 5}
dst := make([]int, len(src))
n := copy(dst, src)
```
