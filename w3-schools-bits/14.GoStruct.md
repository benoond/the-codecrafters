# Go Struct
A **struct** (short for structure) is used to group multiple members of different data types into a single variable.

Unlike arrays, which store multiple values of the same type, structs allow you to combine values of different types.

Structs are particularly useful for organizing related data into a single record.

Declare a Struct

To declare a structure in Go, use the type and struct keywords:

Syntax
```go
type struct_name struct {
  member1 datatype;
  member2 datatype;
  member3 datatype;
  ...
}
```

Example
Here, we declare a struct type `Person` with the following members: `name`, `age`, `job`, and `salary`:

```go
type Person struct {
  name string
  age int
  job string
  salary int
}
```
