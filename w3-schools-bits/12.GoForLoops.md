# Go For Loops
The for loop repeatedly executes a block of code a specified number of times.
In Go, the for loop is the only type of loop available.

The classic for loop has three parts: initialization, condition, and post statement.
Syntax
```go
for initialization; condition; post {
    // code to run repeatedly
}
```
example
```go
package main

import "fmt"

func main() {
    for i := 1; i <= 5; i++ {
        fmt.Println("Number:", i)
    }
}


    i := 1
    for i <= 5 {
        fmt.Println(i)
        i++
    }


     for {
        fmt.Println("Hello Go!")
        break // stops the loop
    }
```
