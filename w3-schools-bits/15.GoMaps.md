# Go Maps
Maps are used to store data as key:value pairs.
Each element in a map consists of a unique key and its corresponding value.
Maps are unordered, changeable collections that do not allow duplicate keys.
The length of a map, or the number of key:value pairs it contains, can be determined using the len() function.
The default value of a map is nil.
Maps store references to an underlying hash table.
Go provides multiple ways to create maps.

Create Maps Using var and :=

Syntax
```go
var a = map[KeyType]ValueType{key1:value1, key2:value2,...}
b := map[KeyType]ValueType{key1:value1, key2:value2,...}
```

Example
```go
This example shows how to create maps in Go. Notice the order in the code and in the output
package main
import ("fmt")

func main() {
  var a = map[string]string{"brand": "Ford", "model": "Mustang", "year": "1964"}
  b := map[string]int{"Oslo": 1, "Bergen": 2, "Trondheim": 3, "Stavanger": 4}

  fmt.Printf("a\t%v\n", a)
  fmt.Printf("b\t%v\n", b)
}
```
