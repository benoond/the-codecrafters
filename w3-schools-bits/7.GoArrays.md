# Go Arrays
Arrays allow you to store multiple values of the same type within a single variable, eliminating the need to declare separate variables for each value.

This example declares two arrays (arr1 and arr2) with defined lengths:
```go
var arr1 = [3]int{1,2,3}
arr2 := [5]int{4,5,6,7,8}
```

## Array Elements
The elements are the individual values stored in the array.
Elements are indexed, usually starting at 0.

This example shows how to access the first and third elements in the prices array:
```go
package main
import ("fmt")

func main() {
  prices := [3]int{10,20,30}

  fmt.Println(prices[0])
  fmt.Println(prices[2])
}
```
