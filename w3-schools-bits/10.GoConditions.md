# Go Conditions
Conditional statements allow a program to execute different actions depending on whether certain conditions are true or false.
A condition can be either true or false.

Go supports the usual comparison operators from mathematics:
- Less than <
- Less than or equal <=
- Greater than >
- Greater than or equal >=
- Equal to ==
- Not equal to !=

Additionally, Go supports the usual logical operators:
- Logical AND &&
- Logical OR ||
- Logical NOT !

You can use these operators or their combinations to create conditions for different decisions.
Example 	Try it
x > y 	
x != y 	
(x > y) && (y > z) 	
(x == y) || z 


## if Statement
The if statement is used to execute a block of code only when a condition is true.
Syntax:
```go
if condition {
    // code to run if condition is true
}

Example:

package main

import "fmt"

func main() {
    age := 20

    if age >= 18 {
        fmt.Println("You are an adult.")
    }
}
```

### if-else Statement
if-else allows you to choose between two paths: one if the condition is true, the other if false.

Syntax:
```go
if condition {
    // code if true
} else {
    // code if false
}

Example:

package main

import "fmt"

func main() {
    age := 16

    if age >= 18 {
        fmt.Println("You are an adult.")
    } else {
        fmt.Println("You are a minor.")
    }
}

```

### else if (optional)
When you have multiple conditions, use else if to check another condition.

Syntax:
```go
if condition1 {
    // code if condition1 is true
} else if condition2 {
    // code if condition2 is true
} else {
    // code if all conditions are false
}
```
