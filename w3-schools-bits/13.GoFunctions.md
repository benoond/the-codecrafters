# Go Functions
A function is a block of code designed to perform a specific task and can be reused throughout a program.

Functions do not run automatically when a program starts; they are executed only when called explicitly.

## Create a Function
To create (or declare) a function, follow these steps:
Use the func keyword.
Give the function a name, followed by parentheses ().
Place the code that defines the function’s actions inside curly braces {}.

Syntax
```go
func FunctionName() {
  // code to be executed
}

```
## Call a Function
Functions are not executed immediately; they are defined and “saved for later use” and will run only when called.

In the example below, we define a function named myMessage(). The opening curly brace { marks the start of the function code, and the closing curly brace } marks its end. This function outputs "I just got executed!". To execute the function, write its name followed by parentheses ():

Syntax

```go
 FunctionName()

```

example

```go
package main
import ("fmt")

func myMessage() {
  fmt.Println("I just got executed!")
}

func main() {
  myMessage() // call the function
}

```
