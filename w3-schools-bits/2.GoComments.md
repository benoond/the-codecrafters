# Go Comments

A comment is a text explain a code and is ignored upon execution.

Comments can be used to explain the code, and to make it more readable.


## Types Comments
The types of comments are
- single-line
- multi-line comments

### Go Single-line Comments
Single-line comments start with two forward slashes (//).
Any text between // and the end of the line is ignored by the compiler (will not be executed).

example
```go
package main //Package declaration

import ("fmt") //Import packages

func main() { //Functions starts
  fmt.Println("Hello World!") //Statements and expressions
} // Functions ends
```
### multi-line comments

Multi-line comments start with /* and ends with */.
Any text between /* and */ will be ignored by the compiler:

example
```go
package main 

import ("fmt") 

func main() { 
  fmt.Println("Hello World!")
} 

/* 
line 1: Package declaration
line 2: Import packages
line 3: Functions starts
line 5: Statements and expressions
line 5: Functions ends
*/
```
