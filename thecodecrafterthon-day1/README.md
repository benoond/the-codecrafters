# Number Converter (Go)

## A simple CLI program that converts numbers between Hex, Binary, and Decimal.

### Features
- HEX → DEC
- BIN → DEC
- DEC → BIN & HEX
### Usage

- Run the program: ie go run main.go 

- Enter a number (or q to quit)
- Choose an operation:
a → HEX to DEC
b → BIN to DEC
c → DEC to BIN & HEX

- The program will display the result and prompt again.

### Notes
Handles invalid input with error messages
Runs in a loop until you quit (q)
